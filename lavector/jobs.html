招聘职位

UI设计师

岗位职责：

  • 1.负责网站静态页设计、制作；
  • 2.负责企业标识、宣传品的设计、制作；
  • 3.根据产品功能，使用流程设计产品网页布局、配色、logo、icon等；
  • 4.配合技术人员及时调整设计，展现最佳效果；
  • 5.协助公司其他部门做设计制作与修改工作。

岗位要求：

  1. 1.2年以上网站美工工作经验；
  2. 2.精通各种美工工具，熟练使用PHOTOSHOP、dreamweaver、illustrator等；
  3. 3.具有较强的美术功底，有设计能力、有成功的网站经验者优先；
  4. 4.工作认真负责、善于沟通，有创新精神；
  5. 5.熟悉WEB产品用户体验理念；
  6. 6.有很强的设计直觉，跟踪最新网站设计趋势。

Web前端工程师

岗位职责：

  • 1. 负责产品软件Web界面的开发；
  • 2. 制定产品Web开发的标准、框架、模板；
  • 3. 确保产品在不同的平台、设备以及浏览器中都有着良好的用户体验；
  • 4. 负责公司网站开发，对网站SEO优化。

岗位要求：

  1. 1. 本科以上学历，有1年以上前台开发实际经验；
  2. 2. 熟练掌握Html5、Javascript、CSS3、jQuery、JSON/REST；
  3. 3. 掌握下列框架之一：AngularJS、Bootstrap、ReactJS、Backbone、Ember等；
  4. 4. 熟悉UI布局、效果实现方法；
  5. 5. 会使用Photoshop进行基本的设计；
  6. 6. 自学能力强，对UI开发有热情。

高级爬虫工程师

岗位职责：

  • 1. 参与核心爬虫系统架构设计、数据库设计；
  • 2. 设计策略和算法，提升网页抓取的数量、效率和质量；
  • 3. 分析爬虫系统的技术缺陷，对策略架构做出合理的调整和改进；
  • 4. 负责社交网站及论坛文章、评论等及时高效高效更新。

岗位要求：

  1. 1. 三年以上Java或python开发经验，熟悉常用的信息抓取策略和数据重组算法；
  2. 2. 熟悉大规模网页爬取，深度网页爬取，熟悉分布式爬虫系统设计，Scrapy、Nutch、Heritrix、WebMagic、等工具者优先；
  3. 3. 熟悉基于Linux/Unix平台的开发，熟悉TCP/IP，HTTP等协议；
  4. 4. 了解Mongodb、HBase、HIVE并至少精通一种关系型数据库的开发；
